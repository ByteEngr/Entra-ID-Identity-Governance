# Entra-ID-Identity-Governance
A collection of sample policies, Graph API automations, and governance workflows for Entra ID.
Entra ID Identity Governance Toolkit

This repository contains demo templates and automation scripts for:

1. Conditional Access

2. Access Reviews

3. Entitlement Management

4. Governance Workflows

5. RBAC and Privileged Identity Management (PIM)

Included:

JSON templates for CA and governance policies

Graph API scripts for automation

Governance architecture diagrams

Documentation for enterprise governance models
